# FalloutRP
## Установка:
1. Устаналиваете SteamCMD по этому гайду: `https://developer.valvesoftware.com/wiki/SteamCMD`
2. Открываете, указываете папку для установки с помощью команды `force_install_dir`
3. Пишете команды: 
4.  1. `login anonymous`
4.  2. `app_update 4020 validate`
5. Пулитесь с `main`
6. Готово!
